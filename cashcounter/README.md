# 🛒 Cash Counter System

A sleek, modern Point of Sale (POS) application built with Python and Tkinter. This application is designed for quick checkout and bill generation, featuring a clean dark-themed interface for menus and a light-themed cart management system.

![Application Preview](https://img.shields.io/badge/Python-3.x-blue.svg)
![Tkinter-based UI](https://img.shields.io/badge/UI-Tkinter-green.svg)

## ✨ Features

- **Dynamic Menu Interface**: Interactive product selection with custom-styled buttons.
- **Real-time Cart Management**: Add items to your cart and watch the totals update instantly.
- **Detailed Invoice Generation**: Professional bill view with individual item pricing and total calculation.
- **Responsive Layout**: Designed to look great on desktop screens (1000x600 resolution).
- **Dark Mode Menu**: A premium aesthetic with high-contrast elements for better readability.

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system.
- Standard libraries included with Python (`tkinter`).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/xAkshat/CashCounter.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CashCounter
   ```
3. Run the application:
   ```bash
   python Cash.py
   ```

## 🛠️ Usage

1. **Add Items**: Click on any product button in the left "MENU" panel.
2. **View Cart**: The right panel displays the items, their unit prices, quantities, and totals.
3. **Generate Bill**: Click the "Generate Bill" button to view a detailed popup invoice.
4. **Clear Cart**: Use the "Clear Cart" button to start a new transaction.
